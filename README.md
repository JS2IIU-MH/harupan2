# Harupan2

某食品メーカー祭りのシールをスマホカメラでかざすだけで自動認識し、合計点数をリアルタイムで計算・表示するアプリです。

## できること

- **シールの自動認識** — カメラにシールをかざすだけで種類と点数を自動判別
- **合計点のリアルタイム表示** — 画面に映ったシールの合計点数をその場で計算
- **SNS シェア** — スクリーンショットと合計点をワンタップで共有
- **前面/背面カメラ切替** — 使いやすい方のカメラで利用可能

## 対応シールと点数

| シール | 点数 |
|:------:|:----:|
| 0.5 点シール | 0.5 点 |
| 1.0 点シール | 1.0 点 |
| 1.5 点シール | 1.5 点 |
| 2.0 点シール | 2.0 点 |
| 2.5 点シール | 2.5 点 |
| 3.0 点シール | 3.0 点 |

## 使い方

1. アプリを起動するとカメラが自動で開始します
2. シールにカメラをかざしてください — 検出されたシールに枠と点数が表示されます
3. 画面中央に合計点数がリアルタイムで表示されます
4. シェアボタンをタップすると、スクリーンショットと合計点を SNS に共有できます

## 対応プラットフォーム

- Android
- iOS
