# CHANGELOG

すべての重要な変更履歴をここに記録します。

## [0.0.1] - 2026-02-22 (Pre-release)

### 追加
- モバイル向け推論アプリの初期骨格（Flutterベース）を追加。
- カメラ入力によるリアルタイム推論パイプラインを実装。
- ONNXモデル `models/best.onnx` を同梱。
- 基本的なUI（カメラビュー、推論結果オーバーレイ、簡易設定）を追加。
- 開発用ユーティリティと統合テストの雛形を追加（`lib/services/image_utils.dart`, `integration_test/` など）。

### 変更
- リポジトリ構成を整理して、ネイティブ側の設定ファイル（`android/`, `ios/`）を含めた状態で提供。

### 既知の問題
- モデル精度は改善の余地あり。環境・解像度に依存して結果が変動します。
- 低スペック端末での推論パフォーマンスが限定的で、フレームドロップが発生する可能性があります。
- バックグラウンド安定性や長時間推論の最適化は未着手。
