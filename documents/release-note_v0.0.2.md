# harupan2 0.0.2

**リリース日**: 2026-02-22

## 概要
このリリース（v0.0.2）は、ユーザーおよび開発者向けに「チュートリアル」を追加した更新です。初めてアプリを触るユーザーが素早く基本操作を理解できる導線と、開発者が環境構築やモデル差し替えを試せるハンズオン手順を提供します。

## 主な追加
- **チュートリアルの追加**: アプリ操作のウォークスルーとドキュメントベースのチュートリアルを追加。
  - 初回起動時のインアプリウォークスルー（UI の基本操作、カメラ起動、推論開始の手順）。
  - 開発者向けセットアップチュートリアル（依存取得、ビルド、モデル差し替え、テスト画像での検証手順）。
  - スクリーンショットとコマンド例を含むドキュメントを `documents/` に追加（参照: 本リリースノート付近のチュートリアル節）。

## 追加・変更点
- チュートリアル関連ドキュメントの追加（手順、スクリーンショット、トラブルシューティング）。
- 初回ユーザー向けの簡易UI説明文言を `lib/widgets/` と `lib/screens/` に反映（軽微な文言調整）。
- ドキュメントに追従して `README.md` の「開発者向け手順」セクションを若干更新（手順の順序・コマンド例の明確化）。

## 既知の問題
- v0.0.1 と同様の既知の問題は引き続き存在します（モデル精度、低スペック端末での処理速度、長時間連続稼働の安定性など）。
- チュートリアルは主要なユースケースをカバーしますが、端末固有のマイクロ調整やパーミッション周りの差異により一部手順が異なる場合があります。

## 動作確認手順（開発者向け）
1. 依存取得: `flutter pub get`
2. デバイスを接続して実行: `flutter run -d <device>`
3. 初回起動時に表示されるチュートリアルを確認し、カメラの使用許可を与えてください。
4. 開発用チュートリアルに従い、必要に応じて `models/best.onnx` を差し替えて検証してください。

## 今後の予定
- チュートリアルの内容拡充（より多くのケース、トラブルシューティングの追加）。
- モデル精度改善と推論最適化。
- ユーザーからのフィードバックを受けてUI・文言の改善を反映。
