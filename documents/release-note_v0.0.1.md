# harupan2 0.0.1 (Pre-release)

**リリース日**: 2026-02-22

## 概要
このプレリリース（v0.0.1）は、現時点での最初の公開バージョンです。モバイル向けに構築された軽量の推論アプリで、カメラ入力からONNXモデルを使った推論を行います。まずは実装済みの機能を手早く検証するためのリリースです。

## 主な機能
- **リアルタイム推論**: カメラ映像を使った推論パイプラインを搭載。
- **ONNXモデル同梱**: `models/best.onnx` をアプリに組み込み済み。
- **クロスプラットフォーム**: Android と iOS 向けに動作（Flutterベース）。
- **基本UI**: カメラビュー、推論結果のオーバーレイ表示、簡易設定。
- **ユーティリティ**: 画像処理用のユーティリティ群（`lib/services/image_utils.dart`）とプロバイダ構成（`lib/providers/app_providers.dart`）。

## 追加・変更点（このリリースで含まれるもの）
- 初期アプリ骨格（カメラ画面、推論ロジック、UIの基本）を追加。
- `assets/` に必要なリソースを配置。
- 組み込みモデル `models/best.onnx` を同梱。
- 統合テストの雛形を `integration_test/` に配置。

## 既知の問題（重要）
- モデルの精度はまだ改善の余地があります。環境や入力解像度で結果が変動します。
- 低スペック端末では推論速度が遅く、フレームドロップが発生する可能性があります。
- バックグラウンド実行や長時間連続推論の安定化は未対応です。
- 多言語対応（UIの翻訳）は限定的です。

## 動作確認手順（開発者向け）
1. 依存取得: `flutter pub get`
2. デバイスを接続して実行: `flutter run -d <device>`
3. Androidの場合は `key.properties` 等の署名設定が必要（リポジトリ内の `android/` を参照）。
4. 同梱モデルは `models/best.onnx`。必要に応じて差し替えて検証してください。

## 開発上の注意点
- 推論エンジンや ONNX ランタイムの設定は `onnxruntime/` や Android/iOS の設定に依存します。ビルド時にネイティブライブラリの組み込みを確認してください。
- テスト用画像は `test_images/` にあります。
