Harupan2 v0.0.2+2 - リリースノート

短い要約:
- 端末上で動作する ONNX ベースの画像推論機能を追加し、安定性と性能を改善しました。

何が新しいか:
- 新機能: カメラ入力を使ったオンデバイス画像推論（ONNX）を追加しました。画像は端末内で処理され、サーバーへ送信されません。
- 改善: 推論速度とメモリ使用量を最適化しました。
- 修正: 一部デバイスで発生していたクラッシュを修正し、安定性を向上しました。

注意事項:
- 本アプリはカメラ権限を使用します。プライバシーポリシーは `documents/privacy_policy.md` を参照してください。

ご意見・不具合の報告はリポジトリの Issues またはサポート連絡先までお知らせください。
